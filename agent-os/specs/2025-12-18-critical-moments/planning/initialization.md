# Feature: Critical Moments (Roadmap Task 1.3)

## Description

Enhance the game's dramatic impact with the following effects:

- **Slow Motion Final Hit**: Time slows to 30% when the KO blow lands.
- **Victory Freeze**: A dramatic pause before the victory pose.
- **Round Start Zoom**: Camera zooms in on fighters before the "Fight!" call.
- **Low Health Visual**: Screen edges pulse red when a player's health is below 20%.
