# Feature Description
Implement Phase 3.1 (Stage Enhancement) and Phase 3.2 (UI Polish) from the roadmap.

## Phase 3.1: Stage Enhancement
- **Parallax Backgrounds**: Multi-layer scrolling backgrounds for depth.
- **Animated Backgrounds**: Moving clouds, swaying trees, ambient characters.
- **Dynamic Lighting**: Stage lighting that reacts to time/attacks.
- **Weather Effects**: Rain, snow, or fog on certain stages.

## Phase 3.2: UI Polish
- **Stylized Health Bars**: Smooth depleting animation with delayed red bar.
- **Character Portraits**: Animated expressions during fight (idle/hit/victory).
- **Round Counter**: Visual indicator of current round.
- **Match Timer**: 99-second countdown with urgency effects.
- **Combo Counter**: On-screen display with flashy animations.
